Finder : Book Search API¶

Finder lets you search the books in bulk via HTTP APIs.

Finder is built on Pyramid. It uses aiohttp to make async calls to the API, making it much faster than simple requests. Finder uses Scout to make book search possible.

Please install Scout manually before proceeding.

Book Search APIs

Request

curl --location --request POST 'http://localhost:6543/search/' \
--header 'Content-Type: application/json' \
--data-raw '{ "limit" : 10, "queries" : ["believe", "changes"] }'
