Install¶

Installation¶

To install Finder, make sure you have the latest stable dist available.

Pre-requisite : Install Scout before proceeding.

Then simply install the current version :

pipenv install "."

Usage¶

pipenv run pserve env.ini

Configuration¶

The Pyramid app can be configured using env.ini configuration file. To learn more about available variables, check Pyramid’s documentation.

Finder app’s custom variables¶

Database is used to specify the default db to query.

database = scout.db
