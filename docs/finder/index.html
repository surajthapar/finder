finder.Indexing¶

Books Indexing Script¶

This script uses Scout to index book summary data. This index is later used for realtime searches.

Make sure you include this script’s relative import path in setup.py;

Specifically, setup –> entry_points –> console_scripts.

You may launch the script by calling it from the virtualenv using pipenv. You must include the configuration file, e.g. env.ini

To install the script,

$ pipenv install -e "."

To run the script,

$ pipenv run index env.ini --corpus data.json

OR

$ pipenv run index env.ini --corpus data.json --database custom.db --slicing [2,5,7]

For help

$ pipenv run index --help
